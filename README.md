
# Personalized Learning System

A data-driven **Personalized Learning System** developed using **Python** and **Jupyter Notebook**.  
This project focuses on analyzing student education data to understand learning patterns and support personalized learning recommendations using data analytics and machine learning techniques.

---

# Repository Structure

```text
personalized_learning_system/
│
├── Personalized_Learning_System.ipynb   # Main Jupyter Notebook
├── student_education_dataset.csv        # Student education dataset
└── README.md                             # Project documentation
````

---

## Project Objectives

* Analyze student education data
* Understand learning patterns and performance
* Apply machine learning models for prediction
* Support personalized learning insights

##  Features

* Exploratory Data Analysis (EDA)

##  Dataset Information

**File:** `student_education_dataset.csv`

The dataset contains information related to students and their academic performance.

### Sample Attributes (example)

| Column Name     | Description                   |
| --------------- | ----------------------------- |
| `student_id`    | Unique student identifier     |
| `gender`        | Gender of the student         |
| `study_time`    | Daily study hours             |
| `attendance`    | Attendance percentage         |
| `math_score`    | Mathematics score             |
| `reading_score` | Reading score                 |
| `writing_score` | Writing score                 |
| `final_result`  | Overall performance indicator |
